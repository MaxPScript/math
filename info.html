Workflow with NTFS junctions (directory symbolic links or technically reparse points)

hard-links-and-junctions
  1. Open cmd on a root? level of the web app.
  2. Type
    mklink /J photos C:\Users\Acer\Documents\my_web_dev_libs\photobank
    The folder "photos" should not exist.
  3. The folder "photos" is now visible at the root level of the web app.
  4. To delete (remove) this junction type
    rmdir photos
